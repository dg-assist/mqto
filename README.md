## Overview

The Manufacturing Quality Troubleshooting Ontology (MQTO) is a domain-independent ontology for manufacturing quality troubleshooting. It provides semantic patterns for connecting quality management knowledge, production context, and data infrastructure to support traceable root cause analysis, failure diagnosis, and countermeasure recommendation.

MQTO occupies the middle layer in a three-layer architecture:
- **Upper layer:** LIS (Lifecycle Information System), SOSA/SSN (Sensor Observation)
- **Middle layer:** MQTO (this ontology) - domain-independent troubleshooting patterns
- **Lower layer:** Domain-specific ontologies (e.g., die-casting, injection molding)

## Key Features

- **Cross-layer integration:** Connects quality management (failures, countermeasures), production context (machines, materials, methods), and data infrastructure (signals, observations)
- **Ishikawa-based organization:** Structures root causes around Machine, Material, and Method categories  
- **Specified/Actual pattern:** Maintains distinction between schema-level definitions and instance-level execution data
- **Standards-based:** Built on LIS, SOSA/SSN, PKO (Procedural Knowledge Ontology), and P-PLAN
- **Modular and extensible:** Designed for easy extension with domain-specific classes and properties

## Core Concepts

### Specified Layer (Schema)
- **IshikawaItem:** Root cause categories (Machine, Material, Method)
- **Failure:** Failure mode definitions with causal relationships
- **Metric:** Key performance indicators with threshold bounds
- **Signal:** Data channel specifications (PLC variables, sensor readings)
- **Countermeasure:** Remediation procedures with ordered steps

### Actual Layer (Instances)
- **ActualIshikawaItem:** Physical machines, material batches, process executions
- **FailureOccurrence:** Concrete failure events
- **MetricAssessment:** Calculated metric values and threshold violations
- **Observation:** Sensor readings and measurements (SOSA)
- **CountermeasureExecution:** Executed remediation actions
