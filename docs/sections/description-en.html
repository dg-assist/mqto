Manufacturing Quality Troubleshooting Ontology: Description back to ToC

Relationship to die-casting ontologies: This ontology (mqto) is a domain-independent generalization of the die-casting quality inspection ontologies. The die-casting Base Model (dg.ttl, namespace https://w3id.org/dgassist#) represents the specified layer for die-casting with domain-specific classes such as Aggregate, Process, Material, and Product. The die-casting Actual Ontology (dga.ttl, namespace https://w3id.org/dgassist/actual#) extends this to the actual/instance layer with observation data and process executions. Both mqto and the die-casting ontologies build upon the LIS (Lifecycle Information System) upper ontology and integrate SOSA/SSN for observations.
